Explanation of operation:
Operation starts with the XAMPP Control Panel, turning on the Apache module with the 
intention to use LOCALHOST.
Then we enter the following address in the web browser: http://localhost/tim3/index.php 
(the zip extraction must be done inside the xamp htdocs folder),
where the index.php file is located that contains the initial structure of our website.
From here we will use the functions requested by the teachers in the browser.
From the form that contains index.php, using method=Post,
They send the data according to the requirements requested in the evaluation.
Explicación del funcionamiento:
El funcionamiento se inicia con XAMPP Control Panel, encendiendo el módulo de Apache con la 
intención de utilizar LOCALHOST.
Luego ingresamos la siguiente direccion en el navegador web: http://localhost/tim3/index.php
